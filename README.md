### 👨‍💻 About Me
Seorang *developer* dan *graphic designer* yang berfokus pada pengembangan komunitas *roleplay* dan ekosistem server (*FiveM / SA-MP*). Saya menggabungkan logika *programming* dengan kreativitas visual untuk membangun identitas komunitas yang kuat dan fitur *in-game* yang fungsional.

*   🎯 **Currently focusing on:** Game server development, UI/UX for in-game assets, and community branding.
*   🎨 **Design Philosophy:** Membawa estetika bergaya *GTA V* dan *streetwear* ke dalam aset digital dan UI *roleplay*.
*   💡 **Fun Fact:** Selain *ngoding*, saya juga merancang *branding* lengkap mulai dari logo geng/fraksi hingga desain *packaging* produk *in-game*.
